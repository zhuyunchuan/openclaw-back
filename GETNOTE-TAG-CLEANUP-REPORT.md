## 📝 标签命名规范（建议）

### 一级分类
```
投资/        - 投资相关
AI/          - 人工智能相关
学习来源/     - 课程、书籍来源
内容类型/     - 笔记类型（系统自动）
```

### 二级分类示例
```
投资/策略     - 投资方法论
投资/工具     - 分析工具、指标
投资/案例     - 具体公司/人物案例
投资/市场     - 市场分析

AI/生成式 AI  - 生成式 AI 相关
AI/具身智能   - 机器人、硬件 AI

学习来源/得到  - 得到课程
学习来源/书籍  - 书籍笔记
```

### 使用规则
1. 每条笔记 2-5 个标签
2. 必须包含 1 个一级分类
3. 人名/公司名作为三级标签（如`投资/案例/安克创新`）
4. 系统标签（AI 链接笔记）自动添加，不手动修改
