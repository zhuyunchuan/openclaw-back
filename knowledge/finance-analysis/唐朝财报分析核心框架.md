# 唐朝财报分析核心框架

> 来源：《手把手教你读财报 2021(新准则升级版)》作者：唐朝

---

## 📌 核心理念

**财报是用来排除企业的**

- 财报不是发现好公司的唯一工具，而是排除坏公司的工具
- 提示风险，引导避开陷阱，而不是保证盈利
- "疑罪从有"原则：对可疑数据保持警惕，宁可错过，不可踩雷

---

## 🎯 企业财务分析四步法

### 第一步：浏览资产负债表和利润表，关注异常

**资产端重点关注：**
- 应收账款
- 应收票据
- 其他应收款
- 长期待摊费用

**负债端重点关注：**
- 应付票据
- 应付账款
- 其他应付款
- 短期借款

**判断标准：** 是否大比例增长（明显超过营业收入增长比例）

**利润表关注：**
1. 营业利润率有没有大比例变化
2. 营业利润中公允价值变动收益、投资收益及资产减值损失三者之和在营业利润中的占比有没有发生重大变化
3. 营业利润与净利润的比例是否保持基本稳定

**处理原则：** 若有异常，必须去附注中寻找解释，甚至致电公司寻求答案。不能获得完整可信解释的，对整份财报就要心存警惕。

---

### 第二步：进行财务指标的历史分析

**核心方法：**
- 连续看 5 年以上数据才能做可信与否的判断
- 警惕上市当年 ROE 较上市前 3 年平均水平下跌超过 50%
- 连续盈利的公司可信度高于连亏后突然盈利的公司

**常见规律：**
- 公司上市当年的 ROE，通常会较其上市前 3 年的平均水平下跌超过 50%
- 以后年度也很难恢复至上市前水平

---

### 第三步：对比企业利润和现金流量变化的趋势

**核心原则：**
- 拉长几年来看，企业的净利润总量与经营活动现金流量净额总量，应该保持一个大致稳定的比例
- **最好是经营活动现金流净额 > 净利润**

**关键问题：** 如果企业获利很好但现金流很差，必须问：企业利润为什么没有转化为现金？

**作用：** 这个思考可以帮助你躲避绝大多数造假企业

---

### 第四步：将企业与竞争对手比较

**比较维度：**
1. **资产负债结构** - 看行业内人士对该企业资源的运用方向是否观点一致
2. **利润表指标** - 通过利润产生过程中的各项收入和费用所占比例的比较，观察管理层是否称职

**警惕信号：**
- 企业增长低于行业平均增长 → 市场份额在降低，竞争优势在消失
- 鸭子没有随着水面上升 → 一定是某处出了问题

## 🛡️ 安全性分析

### 核心指标：现金债务比

**公式：**
```
现金债务比 = (现金及现金等价物 + 可迅速变现的金融资产净值) ÷ 有息负债 > 1
```

**保守版本：**
```
(现金及现金等价物 + 可迅速变现的金融资产净值) ÷ 一年内到期的有息负债 > 1
```

**判断标准：**
- 公司的现金及现金等价物 + 可迅速变现的金融资产（交易性金融资产、银票等）可以覆盖一年内到期的有息负债，就算基本安全
- 无息负债（经营性负债）是企业在产业链强势地位的副产品，只要企业特性没有发生重大不利改变，无须考虑短期偿还

## 📊 盈利能力分析

### 两个角度

**1. 营业收入角度：** 每单位的营业收入有多少可以转化为利润
- 营业利润率
- 毛利率
- 净利率

**2. 资产角度：** 股东投入 + 所借有息负债合计产生多少利润
- 净资产收益率 (ROE)
- 总资产收益率 (ROA)

## ⚠️ 危险信号清单（避雷指南）

### 财务报表异常信号

1. **应收账款/票据异常**
   - 应收账款增长明显超过营业收入增长
   - 应收账款占营业收入比例过高
   - 一年以上应收账款占比过高（如超过三成）

2. **现金流异常**
   - 净利润很好但经营活动现金流很差
   - 净利润总量与经营活动现金流量净额总量比例不稳定

3. **利润结构异常**
   - 营业利润中公允价值变动收益、投资收益及资产减值损失占比发生重大变化
   - 营业利润与净利润比例不稳定

4. **存货异常**
   - 存货增长超过营业收入增长
   - 因产品库存增加造成毛利率提升（需警惕虚增收入）

5. **其他应收款异常**
   - 其他应收款大比例增长
   - 金额过大且无合理解释

6. **固定资产/在建工程异常**
   - 在建工程长期不转固
   - 固定资产折旧方法频繁变更

7. **商誉异常**
   - 商誉占净资产比例过高
   - 商誉减值测试不透明

### 公司经营异常信号

1. **无法按时发布财报**
2. **频繁更换会计师事务所**
3. **供应商或主要客户可疑**
   - 执照没有年审
   - 注册地在居民楼
   - 虚假注册地
   - 税务异常
4. **关联交易频繁且定价不公允**
5. **高管频繁离职**
6. **员工薪酬水平异常**（可帮助排除潜在造假企业）

### 高危行业警示

**造假高发区域：**
- 农、林、牧、渔业
- 生物科技行业

**原因：**
- 生物资产难以盘点（指着 20 万亩湖水说有多少鱼）
- 税收减免政策多，造假成本低

## 🔍 关键财务比率参考

### 安全性指标
| 指标 | 公式 | 健康值 |
|------|------|--------|
| 流动比率 | 流动资产÷流动负债 | ~2 |
| 速动比率 | (流动资产 - 存货)÷流动负债 | ~1 |
| 现金债务比 | 现金及等价物÷有息负债 | >1 |

### 盈利能力指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 毛利率 | (营业收入 - 营业成本)÷营业收入 | 反映产品竞争力 |
| 营业利润率 | 营业利润÷营业收入 | 反映经营效率 |
| 净利率 | 净利润÷营业收入 | 反映最终盈利能力 |
| ROE | 净利润÷净资产 | 反映股东回报 |

## 📝 使用指南

### 分析流程

```
1. 获取财报 → 2. 四步分析法 → 3. 危险信号检查 → 4. 得出结论
                                        ↓
                                   有异常？→ 排除或深入调查
```

### 决策原则

- **疑罪从有** - 对可疑数据宁可相信有问题
- **排除优先** - 先排除坏公司，再考虑好公司
- **历史验证** - 至少看 5 年以上数据
- **现金为王** - 利润必须转化为现金才是真利润
