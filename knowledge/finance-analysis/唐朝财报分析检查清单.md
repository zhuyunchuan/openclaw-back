# 唐朝财报分析检查清单

> 来源：《手把手教你读财报》- 唐朝
> 用途：投资分析助理必备检查清单

---

## ✅ 第一步：快速浏览（5 分钟）

### 资产负债表异常检查

- [ ] 应收账款增长是否超过营收增长？
- [ ] 应收票据是否异常增长？
- [ ] 其他应收款是否大比例增长？
- [ ] 长期待摊费用是否异常？
- [ ] 应付票据/应付账款是否异常增长？
- [ ] 短期借款是否大幅增加？

### 利润表异常检查

- [ ] 营业利润率是否有大比例变化？
- [ ] 公允价值变动收益 + 投资收益 + 资产减值损失 占营业利润比例是否变化？
- [ ] 营业利润与净利润比例是否稳定？

**⚠️ 任何异常 → 去附注找解释 → 解释不可信 → 排除**

---

## ✅ 第二步：历史分析（至少 5 年数据）

### ROE 历史趋势

- [ ] 上市前 3 年平均 ROE 是多少？
- [ ] 上市当年 ROE 是否下跌超过 50%？
- [ ] 上市后 ROE 是否持续低迷？

### 盈利稳定性

- [ ] 是否连续 5 年盈利？
- [ ] 是否有突然扭亏为盈的情况？
- [ ] 盈利增长是否稳定？

**⚠️ 连亏后突然大额盈利 → 高度警惕**

---

## ✅ 第三步：利润与现金流对比

### 核心检查

- [ ] 经营活动现金流净额是否为正？
- [ ] 经营活动现金流净额是否接近或大于净利润？
- [ ] 几年来净利润总量与经营现金流总量比例是否稳定？

### 现金流肖像

- [ ] 三大现金流板块（经营/投资/筹资）正负情况如何？
- [ ] 是否符合企业类型特征？

**⚠️ 利润很好但现金流很差 → 必须找到可信解释**

---

## ✅ 第四步：行业对比

### 增长速度对比

- [ ] 营收增速 vs 行业平均增速
- [ ] 利润增速 vs 行业平均增速
- [ ] 是否跑输行业平均水平？

### 资产负债结构对比

- [ ] 资产负债率 vs 行业平均
- [ ] 应收账款占比 vs 行业平均
- [ ] 存货占比 vs 行业平均

### 盈利能力对比

- [ ] 毛利率 vs 行业平均
- [ ] 净利率 vs 行业平均
- [ ] ROE vs 行业平均

**⚠️ 增长低于行业平均 → 竞争优势在消失**

---

## ✅ 安全性检查（必须通过）

### 现金债务比

```
现金及现金等价物 + 可迅速变现的金融资产净值
--------------------------------------------  > 1
           一年内到期的有息负债
```

- [ ] 现金及等价物是否充足？
- [ ] 可迅速变现的金融资产（交易性金融资产、银票等）有多少？
- [ ] 一年内到期的有息负债是多少？
- [ ] 比率是否 > 1？

**⚠️ 现金债务比 < 1 → 有短期偿债风险 → 排除**

---

## ✅ 危险信号检查（发现即排除）

### 财务报表危险信号

- [ ] 应收账款占营收比例过高或快速增长
- [ ] 一年以上应收账款占比超过三成
- [ ] 存货异常增长
- [ ] 存货增加导致毛利率提升（可能虚增收入）
- [ ] 其他应收款金额过大
- [ ] 在建工程长期不转固
- [ ] 固定资产折旧方法频繁变更
- [ ] 商誉占净资产比例过高
- [ ] 频繁计提或转回资产减值
- [ ] 营业利润主要靠投资收益/公允价值变动

### 公司经营危险信号

- [ ] 无法按时发布财报
- [ ] 频繁更换会计师事务所
- [ ] 供应商/客户注册地在居民楼
- [ ] 供应商/客户执照没有年审
- [ ] 关联交易频繁且定价不公允
- [ ] 高管频繁离职
- [ ] 员工薪酬水平异常（过低或过高）

### 高危行业

- [ ] 是否属于农、林、牧、渔业？
- [ ] 是否属于生物科技行业？
- [ ] 是否有税收减免政策（造假成本低）？

**⚠️ 发现任何危险信号 → 疑罪从有 → 排除**

---

## ✅ 盈利能力评估

### 营业收入角度

- [ ] 毛利率是否稳定或提升？
- [ ] 营业利润率是否稳定或提升？
- [ ] 净利率是否稳定或提升？

### 资产角度

- [ ] ROE 是否 > 15%？（优秀标准）
- [ ] ROE 是否稳定或提升？
- [ ] ROA 是否合理？

---

## ✅ 最终决策

### 通过标准（必须全部满足）

- [ ] 四步分析法无异常
- [ ] 现金债务比 > 1
- [ ] 无危险信号
- [ ] 盈利能力稳定或提升
- [ ] 历史数据可信（5 年以上）

### 决策结果

- ✅ **全部通过** → 可进一步深入研究
- ⚠️ **有疑点但可解释** → 需要更多调查
- ❌ **有危险信号** → 直接排除

---

## 📌 使用原则

1. **疑罪从有** - 宁可错过，不可踩雷
2. **排除优先** - 先排除坏的，再选好的
3. **现金为王** - 利润必须转化为现金
4. **历史验证** - 至少看 5 年数据
5. **行业对比** - 避免被孤立增速迷惑

---

## 🎯 快速决策树

```
开始
  ↓
浏览财报 → 发现异常？→ 是 → 附注解释可信？→ 否 → 排除 ❌
  ↓ 否
历史分析 → 连续 5 年盈利？→ 否 → 警惕 ⚠️
  ↓ 是
现金流检查 → 现金流净额 < 净利润？→ 是 → 找解释 → 不可信 → 排除 ❌
  ↓ 否
安全性检查 → 现金债务比 < 1？→ 是 → 排除 ❌
  ↓ 否
危险信号检查 → 发现危险信号？→ 是 → 排除 ❌
  ↓ 否
行业对比 → 跑输行业？→ 是 → 警惕 ⚠️
  ↓ 否
通过 ✅ → 可深入研究
```

*投资有风险，决策需谨慎。本清单仅供参考，不构成投资建议。*
